# 高中化学核心公式大全
本列表涵盖高中化学**必修+选择性必修**核心公式，按知识点分类整理，标注符号含义，方便复习与查阅。

## 一、物质的量相关公式（高中化学核心基础）
1. **物质的量与质量**
    \[
    n = \frac{m}{M}
    \]
    - \(n\)：物质的量（单位：\(\mathrm{mol}\)）
    - \(m\)：质量（单位：\(\mathrm{g}\)）
    - \(M\)：摩尔质量（单位：\(\mathrm{g\cdot mol^{-1}}\)，数值上等于相对分子/原子质量）

2. **物质的量与微粒数**
    \[
    n = \frac{N}{N_\mathrm{A}}
    \]
    - \(N\)：微粒数（分子、原子、离子等）
    - \(N_\mathrm{A}\)：阿伏伽德罗常数，\(N_\mathrm{A}\approx6.02\times10^{23}\ \mathrm{mol^{-1}}\)

3. **气体物质的量（标准状况）**
    \[
    n = \frac{V}{V_\mathrm{m}}
    \]
    - \(V\)：气体体积（单位：\(\mathrm{L}\)）
    - \(V_\mathrm{m}\)：气体摩尔体积，**标准状况（0℃、101kPa）**下\(V_\mathrm{m}=22.4\ \mathrm{L\cdot mol^{-1}}\)

## 二、气体定律相关公式
1. **理想气体状态方程**
    \[
    PV = nRT
    \]
    - \(P\)：压强（\(\mathrm{Pa}\)），\(V\)：体积（\(\mathrm{m^3}\)），\(T\)：热力学温度（\(\mathrm{K}\)，\(T=t+273.15\)）
    - \(R\)：气体常数，\(R=8.314\ \mathrm{J\cdot mol^{-1}\cdot K^{-1}}\)

2. **阿伏伽德罗定律推论**
    - 同温同压：\(\frac{V_1}{V_2}=\frac{n_1}{n_2}=\frac{N_1}{N_2}\)
    - 同温同体积：\(\frac{P_1}{P_2}=\frac{n_1}{n_2}\)

## 三、溶液浓度相关公式
1. **物质的量浓度**
    \[
    c = \frac{n}{V}
    \]
    - \(c\)：物质的量浓度（单位：\(\mathrm{mol\cdot L^{-1}}\)）
    - \(V\)：溶液体积（单位：\(\mathrm{L}\)）

2. **质量分数**
    \[
    \omega = \frac{m(\text{溶质})}{m(\text{溶液})}\times100\%
    \]

3. **物质的量浓度与质量分数换算**
    \[
    c = \frac{1000\rho\omega}{M}
    \]
    - \(\rho\)：溶液密度（单位：\(\mathrm{g\cdot cm^{-3}}\)）

4. **溶液稀释定律**
    \[
    c_1V_1 = c_2V_2
    \]
    稀释前后溶质的物质的量不变。

## 四、化学反应速率公式
1. **平均反应速率**
    \[
    v = \frac{\Delta c}{\Delta t}
    \]
    - \(v\)：反应速率（单位：\(\mathrm{mol\cdot L^{-1}\cdot min^{-1}}\) 或 \(\mathrm{mol\cdot L^{-1}\cdot s^{-1}}\)）
    - \(\Delta c\)：浓度变化量，\(\Delta t\)：时间变化量

2. **速率与化学计量数关系**
    对于反应\(\ce{aA + bB = cC + dD}\)：
    \[
    v(\ce{A}):v(\ce{B}):v(\ce{C}):v(\ce{D}) = a:b:c:d
    \]

## 五、化学平衡相关公式
1. **化学平衡常数\(K\)**
    对于可逆反应\(\ce{aA + bB \rightleftharpoons cC + dD}\)（固体、纯液体不代入）：
    \[
    K = \frac{[\ce{C}]^c \cdot [\ce{D}]^d}{[\ce{A}]^a \cdot [\ce{B}]^b}
    \]
    - \([X]\)：平衡时物质的量浓度

2. **浓度商\(Q_\mathrm{c}\)**
    表达式同\(K\)，代入**任意时刻**浓度：
    - \(Q_\mathrm{c} < K\)：反应正向进行
    - \(Q_\mathrm{c} = K\)：反应达平衡
    - \(Q_\mathrm{c} > K\)：反应逆向进行

3. **反应物转化率**
    \[
    \alpha = \frac{n(\text{转化})}{n(\text{起始})}\times100\% = \frac{c(\text{转化})}{c(\text{起始})}\times100\%
    \]

4. **产率**
    \[
    \text{产率} = \frac{\text{实际产量}}{\text{理论产量}}\times100\%
    \]

## 六、电离平衡与pH计算
1. **水的离子积常数**
    \[
    K_\mathrm{w} = c(\ce{H+}) \cdot c(\ce{OH-})
    \]
    - 常温（25℃）：\(K_\mathrm{w}=1\times10^{-14}\)

2. **pH与pOH定义**
    \[
    \mathrm{pH} = -\lg c(\ce{H+}) \quad \mathrm{pOH} = -\lg c(\ce{OH-})
    \]
    常温下：\(\mathrm{pH + pOH = 14}\)

3. **弱酸/弱碱电离常数**
    弱酸\(\ce{HA \rightleftharpoons H+ + A-}\)：
    \[
    K_\mathrm{a} = \frac{c(\ce{H+}) \cdot c(\ce{A-})}{c(\ce{HA})}
    \]
    弱碱电离常数\(K_\mathrm{b}\)表达式同理。

4. **盐类水解常数**
    \[
    K_\mathrm{h} = \frac{K_\mathrm{w}}{K_\mathrm{a}(\text{弱酸})} \quad \text{或} \quad K_\mathrm{h} = \frac{K_\mathrm{w}}{K_\mathrm{b}(\text{弱碱})}
    \]

## 七、电化学相关公式
1. **法拉第电解定律**
    \[
    Q = It = nF
    \]
    - \(Q\)：电量（单位：\(\mathrm{C}\)），\(I\)：电流（\(\mathrm{A}\)），\(t\)：时间（\(\mathrm{s}\)）
    - \(n\)：转移电子的物质的量，\(F\)：法拉第常数，\(F=96500\ \mathrm{C\cdot mol^{-1}}\)

2. **原电池/电解池基本规律**
    - 原电池：负极发生**氧化反应**，正极发生**还原反应**
    - 电解池：阳极发生**氧化反应**，阴极发生**还原反应**

## 八、热化学相关公式
1. **反应焓变计算**
    \[
    \Delta H = \text{生成物总能量} - \text{反应物总能量}
    \]
    \[
    \Delta H = \text{反应物键能总和} - \text{生成物键能总和}
    \]

2. **盖斯定律**
    化学反应的焓变只与反应的始态和终态有关，与反应途径无关，可通过热化学方程式叠加计算\(\Delta H\)。
